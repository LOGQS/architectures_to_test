Architectural Properties
Scalable Memory: O(N × d_state) persistent state with O(1) shared parameters
Sparse Computation: Only k active columns per timestep (typically 10-25% of N)
Stateful Processing: Column states persist across sequence positions
Democratic Consensus: Multi-head attention enables cross-column communication
Differentiable Gating: Straight-through estimation for end-to-end training
FiLM Conditioning: Feature-wise modulation integrates consensus messages
Sequential Processing
For input sequence (x₁, x₂, ..., x_T), each timestep updates the persistent column states, enabling long-term memory and temporal reasoning across thousands of specialized micro-modules. The architecture maintains computational efficiency through sparse gating while preserving rich representational capacity through the large column ensemble.
